# Sorting-Visualizer
Sorting Visualizer is a web app for visualizing a bunch of different 

sorting algorithms

Like

• Selection Sort.

• Bubble Sort.

• Insertion Sort.

• Merge Sort.

• Quick Sort.

• Heap Sort.
